How Prometheus was Built

About Prometheus

Prometheus is a NLP model that is able to imitate the writing style and plot of certain novels and scripts
Let's see how it was made!

1: Gathering Data

Book data was gathered from websites like Kaggle.com and scraped data from websites with BeautifulSoup4

2: Training the Model

After cleaning the data with regex, the Python library "aitextgen" was used to train GPT-2

2.5: How it was Trained

Each time the model trains on a line of data, the neural network "backpropagates" and tells itself what parameters to change.
(This results in the Model "learning")

3: Re-training Data

Even after training models for ~100,000 steps, sometimes it would not meet our standards. Cleaning the data better and re-training it helped with accuracy.

4: Saving the Model

To save the parameters set from training the model, files like pytorch.bin, config.json, and tokenizer.aitextgen.json were saved.

5: Sending the Output to You!

When a user clicks on the "Generate" button on this page. Prometheus uses the saved models to finish your prompt and send it back with a POST request.
